## Настройка проекта
1. Установите [pnpm](https://pnpm.io/installation)
2. Установите зависимости проекта:
```bash
pnpm i
```

## Запуск проекта
### Запуск dev сервера:
```bash
pnpm dev
```

### Собрать production build:
```bash
pnpm build
```

### Посмотреть локально production build:
```bash
pnpm preview
```
